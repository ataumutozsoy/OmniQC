## ENGLISH

### Title
OmniQC: A Modern FASTQ Quality Control Application

### Description
OmniQC is a desktop application designed for bioinformaticians to perform comprehensive quality control analysis on FASTQ sequencing files. The application provides an intuitive graphical interface for analyzing next-generation sequencing (NGS) data quality metrics.

**Key Features:**
- Per Base Sequence Quality analysis with Phred score visualization
- Per Sequence Quality Score distribution
- Per Base Sequence Content (A/T/G/C/N) analysis
- GC Content distribution per sequence
- Sequence Duplication Level detection
- Adapter Content detection for Illumina adapters
- N Content analysis for ambiguous bases
- Pass/Warn/Fail quality assessment for each metric
- Professional PDF and CSV report generation
- Project-based sample organization
- Batch analysis with progress tracking

**Technical Stack:**
- Frontend: React, Recharts, TailwindCSS
- Desktop Framework: Electron
- Backend: Python (FASTQ parsing, SQLite database)
- Build System: Vite, electron-builder

**System Requirements:**
- Operating System: Windows 10/11 (64-bit)
- RAM: 4GB minimum, 8GB recommended
- Storage: 500MB

OmniQC: Modern FASTQ Kalite Kontrol Uygulaması

### Aciklama
OmniQC, biyoinformatik uzmanlarinin FASTQ sekanslama dosyalari uzerinde kapsamli kalite kontrol analizi yapabilmesi icin tasarlanmis bir masaustu uygulamasidir. Uygulama, yeni nesil sekanslama (NGS) veri kalitesi metriklerini analiz etmek icin sezgisel bir grafik arayuz sunmaktadir.

**Temel Ozellikler:**
- Phred skor gorsellestirmesi ile Baz Bazinda Sekans Kalitesi analizi
- Sekans Basina Kalite Skor dagilimi
- Baz Bazinda Sekans Icerigi (A/T/G/C/N) analizi
- Sekans basina GC Icerigi dagilimi
- Sekans Duplikasyon Seviyesi tespiti
- Illumina adaptörleri icin Adapter Icerigi tespiti
- Belirsiz bazlar icin N Icerigi analizi
- Her metrik icin Gecti/Uyari/Basarisiz kalite degerlendirmesi
- Profesyonel PDF ve CSV rapor olusturma
- Proje tabanli ornek organizasyonu
- Ilerleme takipli toplu analiz

**Teknik Altyapi:**
- On Yuz: React, Recharts, TailwindCSS
- Masaustu Cercevesi: Electron
- Arka Yuz: Python (FASTQ ayristirma, SQLite veritabani)
- Derleme Sistemi: Vite, electron-builder

**Sistem Gereksinimleri:**
- Isletim Sistemi: Windows 10/11 (64-bit)
- RAM: Minimum 4GB, Onerilen 8GB
- Depolama: 500MB
